# LicenceHub Academy 構想メモ（最新版・価値観アップデート反映）

## 0. 核となる理念（再確認）

**迷わず・続けて・合格できる**

- **学習者の不安を減らす**
- **独学の失敗を「意志」ではなく「構造」で防ぐ**

### 前提の捉え方

- 努力は個人の意志ではなく、設計の問題
- 途中離脱は異常ではなく仕様
- AI 時代の価値は「情報量」ではなく **判断代行・継続支援・意味づけ**

## 1. LHA が提供する価値の中心

### ① 判断代行（迷わせない）

- 今日やることを **最大 3 つ** に絞って自動提示
- 優先順位はシステム側で決定
- 「やらなくていいこと／後回しでいいこと」を明示
- 学習者は **考えずに実行するだけ** の状態を作る

### ② 習熟度の可視化（続けたくなる）

#### 習熟度の定義（アップデート）

- **習熟度とは**
  - 「能力の正確な測定値」ではなく、十分な学習データを根拠に **“今この水準にいると判断できる状態指標”** である。
- 理解できたかを問いすぎない
- 正確さよりも「前に進んでいる感覚」を優先
- 不安を煽らない・責めない指標であることを重視

## 2. 習熟度指標の設計思想

### ① 表現形式

- **0〜100 のメーター表示**
  - 意味：「いまどの辺にいるか」「仕上がり感・到達感」

### ② 習熟度の性質

- 急激に上下しない
- 原則として **減少しにくい**
- 日々の変化は小さく、安心して眺められる

### ③ カテゴリ別習熟度との関係

- 習熟度は **カテゴリ（章・領域）単位** で内部的に算出される
- 全体習熟度は、それらを基にした **集約指標**
  - 単純な平均値そのものを意味しない
- **全体習熟度 ＝**
  - 「複数のカテゴリ習熟度を根拠にした、現在地の要約」
  - カテゴリ別スコアは **根拠**
  - 全体スコアは **結果**

### ④ 根拠の扱い方

- 習熟度の算出根拠は **分解可能**
  - 問題正答率
  - 暗記カード定着度
  - 学習頻度・継続性
- ただし、常に前面には出さない

#### 表示方針

- **ダッシュボード**：習熟度は **1 指標としてシンプルに表示**
- **特設ページ**：
  - カテゴリ別習熟度
  - 影響要因の要約
  - 信頼度（データ十分度）
  - 理解したい人だけが「なぜこの数値なのか」を追える構造にする。

## 3. ノルマ完了トリガー設計（重要）

### 習熟度確定の考え方

- 習熟度は **学習途中では確定しない**
- 「本日のノルマ完了」という **明確な行為** をもって、その日の学習状態をスナップショットとして確定 → **DB に保存**

### 意図

- 評価は常に **事後**
- 学習提案がプレッシャーにならない
- 「やり切った証」として習熟度が増える因果を作る

## 4. 進捗可視化の役割分担

### ① 全体習熟度メーター

- 現在地・仕上がり感
- 状態指標

### ② カテゴリ別習熟度

- なぜその判断なのか
- 原因の可視化

### ③ ノルマ完了履歴（点トラック）

- やり切った日が積み上がる
- 継続の事実を肯定する

**同じデータでも、役割と文脈によって意味を分ける**

## 5. 学習体験ループ（再整理）

1.  **設定**
    - 受験地 / 試験年 / 目標日
2.  **実行**
    - 講義
    - 演習
    - 暗記
3.  **記録**
    - 学習ログ
    - 正誤ログ
    - カード履歴
4.  **分析**
    - Topic 別習熟度
    - カテゴリ別習熟度
5.  **判定**
    - 全体習熟度（0〜100）
    - 信頼度（データ十分度）
6.  **提示**
    - 今日の 3 タスク
    - 翌日の重点
    - 習熟度フィードバック

## 6. AI の位置づけ（再定義）

- **AI は主役ではない**
- 表に出しすぎない
- 役割は翻訳と補助

### AI が担うこと

- 数値 → 意味への変換
- 今日の成長の言語化
- 次の一手の提案補助

**価値の中心は AI そのものではなく、設計と構造**

## 7. ビジネス・運用思想（変更なし／再確認）

- 課金は検証装置
- 無料体験 → 月額 1 プラン
- 解約は隠さない
- 試験後解約は健全
- 防ぐべきは「試験前・直前離脱」
- 戻り道は必ず残す

## 最終まとめ（現在の LHA 像）

- **迷わせない**：判断を代行する
- **責めない**：途中を評価しない
- **続けさせる**：やり切りを可視化する
- **納得できる**：根拠は裏で支える
- **伸ばしていける**：習熟度は積み上がる状態指標
